# Contributing to AI-Agents-From-Zero

感谢你对本项目的关注 🙌

本项目旨在构建一套从零基础到企业级落地的 AI Agent 实战教程。
欢迎所有形式的贡献。

---

## 📌 可以贡献什么？

我们欢迎以下类型的贡献：

- 修复文档错误 / 错别字
- 补充案例说明
- 增加实战项目示例
- 优化代码结构
- 提供更优的 Prompt 模板
- 改进 RAG / Agent 架构方案
- 英文翻译（未来计划）

---

## 🛠 如何贡献？

### 1️⃣ Fork 仓库

点击右上角 Fork

### 2️⃣ 创建新分支

```bash
git checkout -b feature/your-feature-name
```
