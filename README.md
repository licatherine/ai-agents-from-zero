## 📚 课程目录概要

### 01 大模型基础能力构建

| 章节 | 内容概要                                                                                     |
| ---- | -------------------------------------------------------------------------------------------- |
| 01-1 | 大模型（LLM）认识与环境准备：起源与发展、AGI 关系、主流模型特点与适用场景、预训练/微调/推理  |
